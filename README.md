- 현대·기아·제네시스 차량 옵션 DB 설계 및 구축
- Vue 3 Composition API 기반 반응형 프론트엔드
- Node.js / Express REST API 및 MySQL 연동
- Vercel(프론트) + Render Pro(백엔드) 운영

- 다중 조건 검색·필터링 기능 구현 (현대·기아·제네시스 차량)
- Vercel(프론트) + Render(백엔드) 분리 배포 및 성능 최적화
